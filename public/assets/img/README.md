# Pepsi Mood Icons

This folder contains the Pepsi icon used for mood selection.

## Current Files:
- `pepsi-icon.png` - Pepsi logo used for all mood cards

## Customization:
You can replace `pepsi-icon.png` with a custom Pepsi icon or add different variations for each mood if desired.

Recommended size: 32x32px or 64x64px (will be displayed at 32x32px)
Format: PNG with transparent background
